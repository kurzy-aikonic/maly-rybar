# Release Checklist - Maly rybar

## Pred beta releasem

- [ ] MVP flow funkcni bez padu.
- [ ] Data: min. 20 ryb, 150 otazek, 10 uzlu.
- [ ] Paywall zobrazen na premium obsahu.
- [ ] Zakladni analytika odesila eventy.
- [ ] Landing sbira leady.

## Pred public launch

- [ ] Opraveny P0/P1 chyby z bety.
- [ ] Otestovany onboarding, quiz, atlas, denik.
- [ ] Pravni stranky dostupne (privacy, terms).
- [ ] Funnel funguje: landing -> waitlist -> app.
- [ ] Obsahova kontrola fakticke spravnosti.

## Po launchi (72 hodin)

- [ ] Monitorovat padovost a retention.
- [ ] Kontrola konverze landing leadu.
- [ ] Kontrola funnelu Free -> Premium.
- [ ] Prioritizace prvnich oprav a rychly patch plan.
